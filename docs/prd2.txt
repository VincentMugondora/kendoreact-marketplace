## 1. Overview

KendoMarket is a **modern online marketplace web app** built with **React and KendoReact free UI components**.
It allows users to buy and sell products with a **stunning, responsive, and user-friendly UI**.
The app will also feature a **backend API** for handling authentication, product listings, checkout, and order management.

This project is for the **KendoReact Build Without Boundaries Challenge**, showcasing:

* **At least 10 KendoReact components**
* Integration with the **KendoReact AI Coding Assistant**
* Optional **Nuclia RAG integration** for AI-powered product search

* ‚úÖ Use **10+ KendoReact Free Components** (Grid, Button, Input, Dropdown, Form, Chart, Tabs, Modal, Notification, Avatar, etc.)

## 3. Features

### 3.1 User Roles

* **Buyer:** Browse products, search, add to cart, checkout, track orders
* **Seller:** Add/manage products, view sales, respond to buyer messages
* **Admin:** Manage users, products, orders, moderate content

### 3.2 Pages & UI Components

1. **Landing Page** ‚Äì Hero banner, featured products, call-to-action
2. **Marketplace Page** ‚Äì Product grid (Kendo Grid), search (Input + Dropdown), filters (MultiSelect, Checkboxes), pagination
3. **Product Details Page** ‚Äì Tabs for description/reviews, Add to Cart button, Rating display
4. **Cart & Checkout** ‚Äì Cart summary (ListView), Checkout Form (Inputs, Dropdowns, Button, Validation)
5. **User Profile** ‚Äì Dashboard with Tabs for Orders, Wishlist, Account Settings
6. **Seller Dashboard** ‚Äì Product CRUD with Kendo Grid + Modal forms
7. **Admin Dashboard** ‚Äì User & product management with Charts for analytics

### 3.3 Backend Functionality

* Authentication (JWT-based)
* Product CRUD APIs
* Search & Filtering endpoints
* Cart & Checkout endpoints
* Order management
* Messaging system between buyers & sellers
* Admin endpoints

### 3.4 Integrations

* **KendoReact AI Coding Assistant** ‚Äì used to scaffold pages, generate Grid configurations, and validation logic
* **Nuclia RAG (optional)** ‚Äì semantic product search
